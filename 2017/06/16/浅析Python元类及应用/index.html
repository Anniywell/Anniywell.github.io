元编程

元编程我个人理解是：用代码生成（操纵）代码的编程手法。
Python中元编程的几种手段：

  • 预定义方法
  • 函数赋值
  • descriptor
  • eval
  • 元类

上面的几种方式相对于元类来说简单一些，这里暂时先不讲了，以后有时间再去补充

元类

类也是对象

在Python中一切皆对象，同样“类”也是对象

当你在定义一个class时，会在内存中一个对象，对象的名字就是类的名字。
这个对象（类）自身拥有创建对象（类实例）的能力，而这就是为什么它是一个类的原因
